--- MANEIRAS DE ADICIONAR O CSS ---

Inline: Quando os estilos são adicionados por um atributo

Internal: Quando o CSS é adicionado na tag head

External: Quando o CSS é adicionado através de um arquivo externo, depois importado no HTML

--- ANATOMIA DO CSS ---

1 - Nome da Propriedade
2 - Dois Pontos
3 - Valor
4 - Ponto e Vírgula

--- ORDEM CSS ---

Ordem de Aplicação:
inline > internal = external

-- CLASSES E IDS --

Ids: São utilizados para Elementos Únicos
Classes: Servem para um ou mais elementos, geralmente utilizadas em conjuntos de elementos
